Language Switching System

User-friendly language selection:

Language Switcher Implementation
// Language switcher function for Al-Afia
function alafia_language_switcher() {
global $current_lang;
$opposite_lang = ($current_lang == 'en') ? 'am' : 'en';
$lang_text = ($current_lang == 'en') ? 'አማርኛ' : 'English';
$output = '<div class="language-switcher">';
$output .= '<a href="?lang=' . $opposite_lang . '">';
$output .= '<i class="fas fa-language"></i> ';
$output .= $lang_text;
$output .= '</a>';
$output .= '</div>';
return $output;
}
Language Features
  • Session Persistence: Remembers user language preference
  • One-Click Switching: Instant toggle between English/Amharic
